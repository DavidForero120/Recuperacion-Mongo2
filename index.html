Cuestionario

Igual a ($eq)

Este operador coincide con los valores que son iguales al valor dado.

Ejemplo

Si queremos recuperar un documento concreto de la colección de inventario que tenga el valor exacto de cantidad «20», introduciríamos el siguiente comando

db.inventory.find( { qty: { $eq: 20 } } )

Mayor que ($gt)

Este operador coincide si los valores son mayores que el valor dado.

Ejemplo

En este ejemplo, recuperamos los documentos en los que la cantidad es mayor que 15:

db.inventory.find({"qty": { $gt: 15}})

2). Recuperar los documentos de la colección "ficha" en donde la cantidad de aprendices sea mayor a 12

Mayor o igual que ($gte)

Este operador coincide cuando los valores son mayores o iguales que el valor dado.

Ejemplo

recuperamos los documentos cuya cantidad es mayor o igual a 25:

db.inventory.find({"qty": { $gte: 25}})

3). Recuperar los documentos de la colección "nomina" donde la horas trabajadas son iguales o mayores a 48

Menor que ($lt)

Este operador coincide si los valores son menores que el valor proporcionado.

Ejemplo

Busquemos los documentos con una cantidad inferior a 25:

db.inventory.find({"qty": { $lt: 25}})

4). Recuperar los documentos de la colección "jueguetes" en donde la produccion sea menor a 22000

Menor o igual que ($lte)

Este operador sólo coincide si los valores son menores o iguales que el valor dado.

Ejemplo

Busquemos los documentos con una cantidad menor o igual a 25.

db.inventory.find({"qty": { $lte: 25}})

5). Recuperar los documentos de la colección "aprendiz" donde la edad sea menor o igual a 16

No es igual ($ne)

El operador $ne devuelve los documentos en los que el valor especificado no es igual.

Ejemplo

Digamos que queremos seleccionar todos los documentos en los que la cantidad no es igual a 20:

db.inventory.find( { qty: { $ne: 20 } } )
