Comandos de mongo

1. $in: El operador selecciona los documentos donde el
valor de un campo es igual a cualquier valor en la matriz especificada.

Ejemplo: use genero

¿Cual es la no se que?

------------------------------------------------------------------------------

2. $nin: El operador selecciona los documentos cuyo campo contiene
una matriz sin elementos iguales a un valor en la matriz especificada.

Ejemplo: use genero

¿Cual es la no se que?

------------------------------------------------------------------------------

3. $or: Este operador se utiliza para realizar operaciones OR lógicas en la matriz
de dos o más expresiones y seleccionar o recuperar solo aquellos documentos
que coincidan con al menos una de las expresiones dadas en la matriz.

Ejemplo: use genero

¿Cual es la no se que?

------------------------------------------------------------------------------

4. $and: Evalúa una o más expresiones y devuelve verdadero si todas las expresiones
son verdaderas o si se ejecutan sin expresiones de argumento. De lo contrario, $and devuelve falso.

Ejemplo: use genero

¿Cual es la no se que?

------------------------------------------------------------------------------

5. $not: Realiza una operación NOT lógica en la ◃expresión-operador▹ especificada
y selecciona los documentos que no coinciden con la ◃expresión-operador▹.

Ejemplo: use genero

¿Cual es la no se que?

------------------------------------------------------------------------------

6. $nor: Consulta con Dos Expresiones · contener el campo precio cuyo valor
no es igual a 1.99 y contener el campo venta cuyo valor no es igual a verdadero.

Ejemplo: use genero

¿Cual es la no se que?
