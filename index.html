Comandos de mongo

1. $in: El operador selecciona los documentos donde el valor de un campo
es igual a cualquier valor en la matriz especificada.

Ejemplo: db.coleccion.find({campo:{$in:['valor1','valor2',...,'valorN']}});

2. $nin: El operador selecciona los documentos cuyo campo contiene
una matriz sin elementos iguales a un valor en la matriz especificada.

Ejemplo: db.libros.find( { editorial: { $nin : ['Planeta'] } })

Digite la sintaxis correcta que no contenga el nombre 'Pedro',
de la colección estudiantes.

3. $or: Este operador se utiliza para realizar operaciones OR lógicas en la matriz
de dos o más expresiones y seleccionar o recuperar solo aquellos documentos
que coincidan con al menos una de las expresiones dadas en la matriz.

Ejemplo: { $or: [ { valor1 }, { valor2 }, { valor3 } ] }.
