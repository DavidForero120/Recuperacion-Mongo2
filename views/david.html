Comandos de mongo

Preguntas ▼

1. use: Es un comando que se usa para crear una base de datos.

Ejemplo: use perros

2. db: Este comando se usa para ver que base de datos se esta usando.

3. show dbs: Muestra las bases de datos existentes.

4. createCollection: Sirve para agregar una colección nueva.

Ejemplo: db.createCollection("nombreColeccion")

5. show collections: Muestra todas las colleciones dentro de la base de datos que se esta usando.

6. insert: El comando permite agregar informacion dentro de un documento.

Ejemplo: db.perros.insert({nombre: "Lucas", edad: 10})

Ejemplo-insertar varios documentos: db.perros.insert([{nombre: "Lucas", edad: 10},{nombre:"tonny", edad: 3}])

7. find: Trae todos los documentos y si tiene parametro devuelve los que coincidan con el valor

db.perros.find({nombre: "tonny"}) o db.perros.find()

8. update: Sirve para modificar datos de un documento, recive dos parametros uno que busca el documento y el otro el valor a modificar.

db.perros.update({name:"Lucas"}, {$set: {edad: 5}})

9. updateMany: modifica varios los valores de varios documentos

db.perros.updateMany({}, {$set: {edad: 5}})

10. drop: Este comando sirve para eliminar una colección.

db.perros.drop()

11. dropDatabase: Sirve para eliminar la base de datos que esta seleccionada.

db.dropDatabase()
